**《CatBlast》**

《CatBlast》是一款融合了 **三消解谜、角色扮演（RPG）与肉鸽（Roguelike）** 元素的PC独立游戏。在这里，你将带领一支猫猫冒险小队，踏上一场充满未知与挑战的旅程！

#### **核心特色**

* **三消战斗：“绝境”棋盘机制**
  忘掉你对传统三消的认知！在《CatBlast》的战斗中，棋盘上的宝石 **不会主动补充**
  。你的每一步消除都至关重要，既是为了对敌人造成伤害，也是在消耗棋盘上有限的资源。当棋盘上再也没有可以移动消除的组合时，就会进入“绝境”状态——敌人将根据剩余宝石的数量发动毁灭性的反击！你需要巧妙规划，或利用强大的技能打破僵局，将危机转化为胜机！

* **每一次冒险都是独一无二的肉鸽体验**
  你将在一张随机生成的地图上前进，自由选择你的路线。每一个节点都可能是一场激烈的战斗、一次奇妙的相遇、一个神秘的商店，或是一个奖励丰厚的宝箱。你会遇到能改变战局的强大
  **“遗物”**，招募到新的伙伴，或是面临艰难的抉择。没有两场冒险是完全相同的，每一次出发都充满了新鲜感。

* **组建你最强的小队！**
  收集并培养数十位个性鲜明的猫猫角色！每位角色都拥有独特的定位（如 **输出、坦克、辅助**
  ）和专属技能。通过消除特定颜色的宝石来为他们的技能充能，并释放华丽的终结技。探索角色之间的羁绊与组合，搭配出攻无不克的完美编队，以应对越来越强大的敌人。

* **深度策略与丰富的RPG养成**
  根据敌人的特性，合理搭配你的队伍阵容和技能组合。战斗胜利后，你可以为你的角色 **升级技能、穿戴装备**
  ，让他们变得更强。如何分配资源，选择哪条成长路线，将是你在冒险中必须不断思考的课题。
